# v26.51.19 (2026-06-12)

## Feat: PaddleOCR v2 Job API 迁移

PaddleOCR 从旧版同步 API 升级到 v2 异步 Job API，统一调用方式。

### 变更

- **删除旧 API 端点**（同步，各模型独立 URL）：
  - `PADDLEOCR_OCR_API_URL`
  - `PADDLEOCR_VL_API_URL`
  - `PADDLEOCR_VL15_API_URL`
- **新增统一端点**：`PADDLEOCR_JOB_API_URL`（v2 异步 Job，所有模型共用）
- **更新默认模型**：`pp_ocrv5` → `PP-OCRv6`
- **更新 Token 格式**：`Authorization: token {TOKEN}` → `Authorization: bearer {TOKEN}`

### 数据库迁移

- `core/migrations/0012_migrate_paddleocr_v2_job_api.py`
