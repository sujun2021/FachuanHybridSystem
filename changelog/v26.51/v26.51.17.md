## 前端

### 修复
- **案件编辑页当事人下拉框显示为空**：修复 `admin_case_form.js` 中 `updateClientOptions` 修改 `<select>` option 后 Select2 不同步的问题
  - 根因：Django admin 通过 `jQuery.noConflict(true)` 将 jQuery 移入 `django.jQuery` 命名空间，原代码检查 `typeof jQuery` 永远为 undefined，`refreshSelect2` 从未执行
  - 修复：用 `window.django.jQuery` 获取 jQuery 实例，调用 `select2('destroy')` + `select2()` 销毁并重建 Select2 实例，强制重新读取 DOM 中的 option
  - 影响范围：`updateClientOptions`、`restoreAllClientOptions`、`fillSelect` 三处 DOM 操作后均调用 `refreshSelect2`
