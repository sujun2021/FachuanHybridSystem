### TypeScript 6.x 升级说明

- 移除 `tsconfig.json` 和 `tsconfig.app.json` 中废弃的 `baseUrl` 选项
- `paths` 别名自动相对于 tsconfig 文件位置，无需 baseUrl
- 编译和构建均无报错

### Vite 8.x 升级说明

- Vite 8 使用 Rolldown 替代 Rollup 作为打包器
- `manualChunks` 从对象格式改为函数格式（Rolldown 要求）
- 构建速度提升：约 6s → 1.5s

### ky 2.x 升级说明

- API 无破坏性变更
- TypeScript 编译和 Vite 构建均通过
