# v26.53.0 (2026-06-18)

## 新增：OnlyOffice DocSpace 云文档编辑接入

接入 OnlyOffice DocSpace 云版，提供在线创建、编辑、预览、上传、下载 Word/Excel/PPT 文档的能力。采用 iframe 直接嵌入编辑器，无需额外 SDK 依赖。

---

## 架构设计

```
前端（iframe 嵌入）          后端（API 代理层）           DocSpace 云
    │                            │                          │
    ├─ 新建/上传 ──────────────→ 转发请求 ────────────────→ 创建文件
    │                            ← 返回 webUrl + 文件信息 ←─
    │                            │                          │
    ├─ <iframe src={webUrl}> ───┼─────────────────────────→ 编辑器
    │                            │                          │
    ├─ 列表/下载/删除 ─────────→ 代理 DocSpace API ──────→ 文件操作
```

- 后端：`apps/docspace/` — DocSpace REST API 客户端 + Django Ninja 路由
- 前端：`features/docspace/` — 可复用组件（DocSpaceFrame、DocumentList、UploadButton）
- 配置：Django admin → 系统配置 → DocSpace 分类（Portal URL + API Token）

### API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/docspace/config` | 返回 portalUrl + enabled 状态 |
| POST | `/docspace/upload` | 上传文件到 DocSpace |
| POST | `/docspace/create` | 新建空白 .docx 文档 |
| GET | `/docspace/documents` | 列出当前用户的文档 |
| GET | `/docspace/documents/{id}` | 获取文档详情（含编辑器 URL） |
| DELETE | `/docspace/documents/{id}` | 删除文档（远端 + 本地） |
| GET | `/docspace/documents/{id}/download` | 下载文档 |
| POST | `/docspace/sync/{id}` | 刷新文档元数据 |

### 配置管理

- SystemConfig 新增 `DOCSPACE` 分类
- `DOCSPACE_PORTAL_URL` — Portal 地址（必填）
- `DOCSPACE_API_TOKEN` — API Token，is_secret=True 加密存储（必填）
- 上传文件夹 ID 自动从 DocSpace API 获取当前用户的「我的文档」，无需手动配置
- 只要 Portal URL 和 API Token 配置完成，功能自动启用

## 关键设计决策

| 决策 | 理由 |
|------|------|
| iframe 直接嵌入而非 SDK | SDK 在 localhost 环境下 CSP 校验失败，iframe 更简单可靠 |
| folder ID 自动发现 | 不同用户的「我的文档」ID 不同，通过 API 自动获取 |
| 配置通过 SystemConfig 管理 | 复用已有配置体系，admin 可视化管理，密钥加密存储 |
| get_or_create 处理文件去重 | DocSpace 对相同内容做去重，返回已有 file_id |
| DocSpaceDocument 独立模型 | 一期不做业务关联，二期按需加 ForeignKey 到 Case/Contract |
